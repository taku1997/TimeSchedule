# サークル用に作成した時間割管理アプリ

## 概要
サークルの幹事を担当していた経験から，施設管理の杜撰さに気づき<br>
自分のプログラミング技術で解決できないかと思い，作成したWebアプリケーション<br>

## 使用言語
Javascript(Node.js),HTML(Pug),CSS,PostgreSQL

## 機能一覧
・ログイン・登録機能<br> 
・タイムテーブル形式の時間割管理<br>
・施設の使用登録＆解除<br>

## 注力した機能
・モーダルウィンドウやシングルページなどの理解しやすいデザインや機能<br>
・時間割形式のデータ保持，個人ユーザーの編集権限<br>
